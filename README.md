## Quick start

To install & use the package, install [Julia](https://julialang.org/install/)
then type, within a Julia session:
```julia-repl
julia> using PhyloSummaries
```
The first time, julia will prompt you for package installation.

Then, follow PhyloSummaries's [latest documentation][docs-dev-url].
You may also find PhyloNetworks'
[documentation](https://juliaphylo.github.io/PhyloNetworks.jl/latest/) helpful.
